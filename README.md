# 📊 Sales Performance Dashboard
Built with Power BI | 2022–2024 Data | Indian Retail

---

## 📌 Project Overview
A 1-page interactive Power BI dashboard analyzing ₹1,638.7M in sales across 3 years for an Indian retail business covering 5 regions, 8 categories and 15+ states.

---

## 📊 Key Metrics

| Metric | Value |
|---|---|
| Total Revenue | ₹1,638.7M |
| Gross Profit | ₹697.3M |
| Profit Margin | 42.6% |
| Total Orders | 12,000 |
| Return Rate | 7.6% |
| Years Covered | 2022, 2023, 2024 |

---

## 🔍 Key Business Insights

| Insight | Finding |
|---|---|
| Top Region | North India — ₹493.7M |
| Top Category | Electronics — ₹989.4M (60% of revenue) |
| Top State | Madhya Pradesh — ₹159.87M |
| Top Channel | Online App — ₹490.8M |
| Seasonal Pattern | Q4 (Oct–Dec) peaks every year consistently |
| Margin Stability | Gross margin stable at 42.6% across all years |

---

## 🛠️ Tools Used
- Power BI Desktop
- DAX (Calculated Measures)
- Power Query
- CSV Dataset

---

## 📂 Dataset
- Source: Custom Indian Retail Dataset
- Size: 12,000 rows | 25 columns
- Fields: Region, State, Category, Sub-Category, Channel, Segment, Net Sales, Gross Profit, Returns

## 🚀 How to Use
1. Download `Sales_Performance.pbix` and open in Power BI Desktop
2. Use **Year Slicers** (top right) to filter by 2022 / 2023 / 2024
3. Click any visual to **cross-filter** the entire dashboard
